## **Getting Started**

### **Clone repository**

    $ git clone https://github.com/cassjs/venture-insight.git
    
Navigate to the project folder

      $ cd venture-insight

### **MacOS Setup**

* Install Django

      $ pipenv install django  

* Activate Enviornment

      $ pipenv shell
      
* Run Application

      $ ./runserver.sh
      
### **Windows Setup**
      
* Install Django

* Activate Enviornment
